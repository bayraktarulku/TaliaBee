BROADCOM BCM 2835
=================

bcm2835 is a C library for Raspberry Pi (RPi). It provides access to GPIO and
other IO functions on the Broadcom BCM 2835 chip, as used in the RaspberryPi,
allowing access to the GPIO pins on the 26 pin IDE plug on the RPi board so you
can control and interface with various external devices.

Download link
=============
[http://www.airspayce.com/mikem/bcm2835](http://www.airspayce.com/mikem/bcm2835/)

Installation
============
Download the latest version and put the `.tar.gz` file into
`component/bcm2835/` directory.
